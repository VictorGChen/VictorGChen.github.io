# Regular Project

This is an example of a regular project in the "all" category.

## How Projects are Categorized

Projects are organized by:
- **Category**: Set with `category:` (featured, all, work, fun, etc.)
- **Importance**: Lower numbers appear first (1, 2, 3...)

## To Add Your Own Projects

1. Create a new `.md` file in the `_projects/` folder
2. Copy the front matter from this example
3. Customize it for your project
4. Write your content below the front matter

You can delete this example file when ready!
